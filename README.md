# WYMusic-H5
---
基于`JQuery`的仿网易云音乐的Web端界面

## 功能介绍
- 网站数据的展示, 歌单列表, 歌曲排行榜的数据展示
- 响应式布局, 对于不同屏宽下的布局调整
- 音乐的播放, 搜索, 下载

## 使用
1. 直接在浏览器打开根目录下的index.html文件即可
2. 部分数据因为接口被加密使用的固定json, 部分请求接口放在api.php中, 运行这个php文件然后将setup.js文件中的apiUrl参数的地址改为你当前电脑的IP
